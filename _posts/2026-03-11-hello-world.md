# My First Blog

Hi, I have decided to make a blog. This will be the place documenting all of my projects, publications (hopefully soon), and maybe some random blogs. This website was made in 30 minutes using Jekyll's Chirpy theme. Although it requires me to use Visual Studio and GitHub every time I want to write something (which is not really your typical blogging experience), I found making a github.io site quite intuitive and not as complex as I thought it would be.

# Why I Decided to Use a Chirpy Template

All of my lecturers use Google Sites for their personal websites, which—I get it—is simple to set up and doesn't take much time. However, it does not support post categorising, archiving, etc. And if I were to make a personal website, I would envision it to be kind of like a central hub where, apart from my publication work, I can sometimes write blogs about something I did over the weekend, for example. I could have used the free WordPress plan, but it is very laggy in the browser and takes a lot of time to set up.

Then I remembered that someone I used to know had a site that ended with github.io, and since I have been getting familiar with how GitHub works through my personal projects and dissertation, I thought I should give it a try. And here we are.
